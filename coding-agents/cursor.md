- Cursor supports all frontier coding models from OpenAI, Anthropic, Google, and more.
  - Sonnet 4.6 is Anthropic's medium-tier intelligence model.
  - Claude Opus 4.8 is Anthropic's strongest model and a meaningful jump over Opus 4.7 on CursorBench.
  - Gemini 3.1 Pro is Google's latest model. It processes images alongside code, making it strong for UI/UX work from design mockups. 
  - Gemini 3.5 Flash is Google's newest speed-tier model. It sits between Gemini 3 Flash and Gemini 3.1 Pro on price.
  - GPT-5.5 is OpenAI's most intelligent model in Cursor.
  - GPT-5.3 Codex is OpenAI's flagship coding model.
  - Grok Build 0.1 is xAI's fast coding model trained specifically for agentic coding.
  - Composer 2.5 is Cursor's own agentic model.
 
- Cursor IDE vs Agents window
  - https://cursor.com/docs/agent/agents-window
  - In Cursor, the default IDE experience and an Agent window are related but serve different purposes.
    - Default Cursor IDE = "pair programmer."
    - Agent Window = "junior engineer who can take a ticket and work through it."
